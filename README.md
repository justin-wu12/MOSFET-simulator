# MOSFET-simulator
繁體中文：一個基於 React 的互動式 MOSFET 物理模擬器。視覺化呈現通道形成、夾止現象與 I-V 特性曲線，並支援通道長度調變 (CLM) 效應與小訊號參數 ($g_m, r_o$) 的即時計算。English:A React-based interactive MOSFET physics simulator. Visualizes channel formation, pinch-off, and I-V characteristic curves. Supports Channel Length Modulation (CLM) effects and real-time calculation of small-signal parameters ($g_m, r_o$).
